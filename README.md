# -Python
Ejercicios de nivelacion de conceptos basicos.

La actividad tiene como objetivo practicar algunos conceptos básicos del lenguaje.
Cada ejercicio está hecho en un archivo diferente de Python.

## Estructura del proyecto

Los archivos del proyecto están organizados así:
actividad-python
ejercicio1.py
ejercicio2.py
ejercicio3.py
ejercicio4.py
ejercicio5.py


### Descripción de los ejercicios

**Ejercicio 1**
Se crea una estructura para representar un estudiante con su nombre, edad, estado, materias y notas.
El programa calcula el promedio, la materia con mejor nota, la de peor nota y si el estudiante aprueba o no.

**Ejercicio 2**
El programa pide un valor monetario y calcula:

* el impuesto del 19%
* un descuento del 10% si el valor es mayor o igual a 200000
* el total final.

**Ejercicio 3**
Se generan valores de la serie Fibonacci y se guardan en una lista.
Al final se muestra la lista completa, la cantidad de términos y el último valor generado.

**Ejercicio 4**
El usuario puede elegir entre tres opciones:

1. Saber si un número es primo
2. Calcular el factorial de un número
3. Contar cuántas vocales hay en un texto

**Ejercicio 5**
Se crea una función para recibir nombre, edad y correo.
El programa valida que los datos sean correctos antes de guardarlos en una lista.

## Cómo ejecutar los ejercicios
Para ejecutar los programas necesitas tener **Python 3 instalado**.
Desde la terminal, ubícate en la carpeta del proyecto y ejecuta el ejercicio que quieras probar.

## Convención de nombres
En los archivos se utilizó la convención snake_case, que es la más común en Python.
Ejemplo:
calcular_promedio
contar_vocales
validar_correo
